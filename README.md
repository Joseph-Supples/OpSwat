# Joseph Supples OpSwat MetaDefender File Hash Lookup and Upload
Run via
> node src/index.js

## Description
This uses an API key generated from MetaDefender to access the public API. Generate your own API keyand store it in a .env file in the root directory. The file used for this demonstration is stored as 'sample.txt' in the root directory. If you would like to use a different sample file, update the references.
